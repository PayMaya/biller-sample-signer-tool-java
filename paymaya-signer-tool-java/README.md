# README - PAYMAYA SIGNER TOOL JAVA v1.0

##Pre-requisites:
* Must have JDK installed in your machine
* Must have Gradle installed (use latest version available)

##Steps to Run:

1. Checkout of uncompress the project zip

2. Initialize Gradle

    <code>gradle init</code> 

3. Build the project

    <code>gradle build</code>

4. Run the project

    <code>gradle run</code>

5. Verify output

<code>Task :run
Computed Paymaya Signature: D+IKW5fHA4E4YiKAy/3nxDtsTbYVds0rAzFNNz3bSM4=</code>
